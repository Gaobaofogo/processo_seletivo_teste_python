# Repositório para processo seletivo

## Como rodar o projeto

Todo o código fonte da tarefa se encontra no arquivo `main.py`. Foi utilizado o python 3.14 e todas as dependências se encontram no arquivo `requirements.txt`. Com o Python 3.14 disponível, deve-se rodar:

```bash
$ pip install -r requirements.txt
```

E para rodar o código:

```bash
$ python main.py
```

## Estrutura do arquivo e como ler as questões

As 4 questões estão separadas dentro do arquivo `main.py`. As questões tem a estrutura de uma explicação inicial antes do código, a resolução da questão e no fim existem `assert` para realizar os testes e validação do código produzido com o resultado esperado. Ao longo das questões utilizei mais de um método de estrutura de dados do python, exibição de erros sinalização de erro e solução de algoritmo na questão 3 como forma de exemplificar as muitas possibilidades que a linguagem traz. Deve-se levar em consideração que em produção cada caso é um caso e é necessário ponderar eficiência e custo de manter aquela unidade de código com seu respectivo propósito.
